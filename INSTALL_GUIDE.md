# Инструкция по установке приложения на телефон

Да, вы можете установить приложение на свой телефон **без публикации в Google Play**! Есть несколько способов:

## Способ 1: Debug APK (Самый простой)

### Шаг 1: Соберите Debug APK

Выполните одну из команд:

```bash
# Используя npm скрипт
npm run build:android:debug

# Или напрямую через Gradle
cd android && ./gradlew assembleDebug
```

После сборки APK файл будет находиться здесь:
```
android/app/build/outputs/apk/debug/app-debug.apk
```

### Шаг 2: Перенесите APK на телефон

**Вариант A: Через USB кабель**
1. Подключите телефон к компьютеру через USB
2. Скопируйте файл `app-debug.apk` на телефон (в любую папку, например Downloads)

**Вариант B: Через облачное хранилище**
1. Загрузите `app-debug.apk` в Google Drive / Dropbox / Яндекс.Диск
2. Откройте на телефоне и скачайте файл

**Вариант C: Через ADB (если телефон подключен)**
```bash
adb install android/app/build/outputs/apk/debug/app-debug.apk
```

### Шаг 3: Установите на телефоне

1. Откройте файловый менеджер на телефоне
2. Найдите файл `app-debug.apk`
3. Нажмите на него для установки
4. Если появится предупреждение о неизвестном источнике:
   - Перейдите в **Настройки → Безопасность → Неизвестные источники**
   - Разрешите установку из неизвестных источников
5. Нажмите "Установить"

## Способ 2: Прямая установка через ADB

Если телефон подключен к компьютеру через USB с включенной отладкой по USB:

```bash
# Убедитесь, что Metro bundler запущен (в отдельном терминале)
npm start

# В другом терминале выполните:
npm run android
```

Это автоматически соберет и установит приложение на подключенный телефон.

## Способ 3: Release APK (Для более стабильной версии)

Для сборки Release APK нужно настроить keystore. Сейчас в проекте есть настройка для release, но нужен keystore файл.

### Сборка Release APK:

```bash
npm run build:android:release
```

APK будет здесь:
```
android/app/build/outputs/apk/release/app-release.apk
```

## Важные замечания

⚠️ **Debug APK**:
- Подходит для тестирования
- Больше размер файла
- Может работать медленнее, чем release версия

✅ **Release APK**:
- Оптимизирован для продакшена
- Меньше размер
- Быстрее работает
- Требует настройки keystore для подписи

## Разрешение установки из неизвестных источников

На Android 8.0+ нужно разрешить установку из неизвестных источников:
1. **Настройки → Приложения → Специальный доступ → Установка неизвестных приложений**
2. Выберите файловый менеджер, который вы используете
3. Разрешите установку приложений

## Troubleshooting

**Проблема**: "Приложение не устанавливается"
- Проверьте, что разрешена установка из неизвестных источников
- Убедитесь, что на телефоне достаточно места
- Попробуйте удалить предыдущую версию приложения, если она была установлена

**Проблема**: "APK не найден"
- Убедитесь, что сборка завершилась успешно
- Проверьте путь: `android/app/build/outputs/apk/debug/app-debug.apk`

**Проблема**: "Ошибка при сборке"
- Убедитесь, что установлен Android SDK
- Проверьте, что все зависимости установлены: `npm install`
- Для Android может потребоваться: `cd android && ./gradlew clean`
